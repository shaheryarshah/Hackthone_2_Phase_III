# Claude Code CLI - Documentation Skills

## Overview

This document details the documentation-specific skills within the Claude Code CLI agent system. These skills are primarily utilized by the Documentation Agent but may be leveraged by other agents when documentation tasks are required.

## Skill Directory

### 1. README Management
**Description**: Update README files and project documentation
**Agent**: Documentation Agent
**Capabilities**:
- Maintain project overview and setup instructions
- Keep usage instructions current
- Document project architecture and components
- Update contribution guidelines

**Files & Directories**:
- `README.md`
- `RAG_CHATBOT_README.md`

**Best Practices**:
- Keep README files current and accurate
- Include clear setup instructions
- Document project architecture
- Provide contribution guidelines
